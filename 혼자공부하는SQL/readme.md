# DBMS
- 데이터의 집합인 데이터베이스를 관리하고 운영하는 소프트웨어
- 다양한 데이터가 저장되어 있고 데이터베이스는 여러 명의 사용자나 응용 프로그램과 공유하고 동시에 접근이 가능해야 한다.

종류: MySQL, 오라클, SQL 서버 등
- DBMS에 데이터를 구축, 관리하고 활영하기 위해서 사용되는 언어가 SQL(Structured Query Language).
